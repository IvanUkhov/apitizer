# Apitizer
The main ingredient of a RESTful API client.

## Installation
Add the following line to your `Gemfile`:
```ruby
gem 'apitizer'
```

Then execute:
```bash
$ bundle
```

Alternatively, you can install the gem manually:
```bash
$ gem install apitizer
```
